# AI 兼人工 客服系统 · PRD 总览

> 版本: v1.0  ·  日期: 2026-04-27  ·  状态: Draft

## 1. 产品定位

为 **直播 / 点播** 业务量身打造的「AI 优先 + 人工兜底」客服系统,主打 **高可用、高并发、强业务上下文**。核心目标:

- **行业贴合**: 深度理解直播/点播场景 — 卡顿/黑屏/无声、清晰度切换、CDN 故障、会员订阅、虚拟币与打赏、弹幕/礼物、UGC 举报、版权下架、主播咨询、未成年退款 等。
- **AI 精准拦截**: 高频、标准化、低风险问题由 AI 一次性解决,目标拦截率 ≥ 70%(直播间内简单咨询目标 ≥ 80%)。
- **强上下文传递**: 用户进入客服时自动带入「当前观看节目 / 直播间 / 主播 / 设备 / 清晰度 / CDN 节点 / 最近卡顿事件」,首条消息即"对症下药"。
- **人工无缝兜底**: 当 AI 置信度不足、用户主动转人工或涉及合规/退款/未成年场景时,无感切换到人工座席,且上下文/播放诊断完整透传。
- **现代化体验**: C 端遵循 Apple HIG 最新规范 + Glassmorphism,在 PC、移动 H5、App WebView(直播间内悬浮 / 全屏横屏 / PIP)中均保持一致质感且不阻挡播放。
- **可演进架构**: 支撑后续多渠道(微信、抖音、小程序、TV / OTT)、多租户(SaaS)、多模态(语音、视频通话客服)扩展。

## 2. 用户角色

| 角色 | 描述 | 主要诉求 |
|------|------|----------|
| C 端用户 | 直播观众 / 点播会员,通过 Web H5 / 直播 App WebView 接入 | 卡顿快速定位、订阅/退款/会员、举报、主播相关咨询 |
| 主播 | 入驻主播 | 开播工具、礼物提现、限流/封禁申诉、内容合规咨询 |
| 人工座席 | 客服坐席(分播放/会员/内容/主播组) | 高效处理 AI 兜底会话、查看播放诊断与会员上下文、批量协作 |
| 客服主管 | 班组长 / 运营 | 排班、监控 SLA、质检、知识库治理、突发事件指挥 |
| 系统管理员 | 平台/租户管理员 | 权限、渠道接入、AI 模型与知识库、CDN/IM/礼物系统对接 |
| AI 训练师 | 知识库维护者 | 语料标注、Prompt 调优、FAQ 治理、效果回归 |
| 风控/合规 | 内容安全 / 法务 | UGC 举报溯源、未成年退款、版权下架处置 |

## 3. 核心目标 (Success Metrics)

| 指标 | 目标 |
|------|------|
| AI 一次性解决率 | ≥ 70%(直播间内简单咨询 ≥ 80%) |
| 平均首响 (人工) | ≤ 15s |
| 转人工成功率 (上下文完整) | 100% |
| 播放诊断附带率(播放类工单) | ≥ 95% |
| 系统可用性 SLA | 99.95% |
| 单实例并发会话 | ≥ 5,000 长连接 |
| 全局并发会话 (集群) | ≥ 500,000 |
| 大主播开播突发承载 | 5min 内会话量 10x 不降级 |
| 消息端到端延迟 (P95) | ≤ 300ms |
| C 端首屏 LCP | ≤ 1.5s (4G,WebView 悬浮) |

## 5. 范围与非范围

**In Scope (V1)**

- C 端 Web H5 (PC + Mobile + WebView 自适应,含直播间悬浮入口/全屏横屏/PIP 共存)
- 直播/点播业务上下文注入(节目、直播间、主播、清晰度、CDN、设备、网络、卡顿事件)
- 一键播放诊断上报(自动附带最近 60s 播放日志 / QoE 数据)
- AI 中枢 (LLM + RAG + FAQ + 工具调用 + 转人工策略;默认 ChatGPT API,Key/模型可后台配置)
- 领域工具:`get_play_diagnostics` / `get_membership` / `get_room_info` / `get_vod_info` / `cancel_subscription` / `report_content` 等
- 人工座席工作台 (Web,带播放诊断 / 会员订阅 / 直播间快照 侧栏)
- 管理后台 (知识库、常见问题(多级)、坐席、技能组、公告、快捷语、AI 模型档位、业务系统对接)
- 消息类型: 文本 / 图片 / 文件 / 播放诊断卡片 / 直播间快照卡片
- 顶部公告跑马灯、输入框上方快捷按钮、欢迎语后多级常见问题

**Out of Scope (V1)**

- 原生 App SDK (iOS/Android)、小程序、TV / OTT 端
- 第三方渠道(企微、抖音、视频号)
- 语音/视频通话客服、屏幕共享
- 多租户 SaaS 计费
- 离线工单系统(仅预留接口)
- 主播侧专用工作台(主播咨询暂复用 C 端,V2 再独立)
